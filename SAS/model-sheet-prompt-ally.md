# Character Model Sheet Prompts — Ally Quinn (master template)
*v1 · 2026-07-11 · Attach the Ally studio portrait (microphone/lighthouse image) as the reference with EVERY prompt below.*
*This document doubles as the template for all future character sheets — swap the canon block and rerun.*

## How to use

1. Run **Prompt 0 (the master sheet)** first. It produces the single-page overview.
2. Then run **Prompts A–E** to produce each section at full resolution — these are the working documents artists and AI generations will actually reference.
3. Check every output against the **acceptance checklist** before saving it into canon. The first approved set becomes Ally's permanent reference; every future generation attaches these sheets, not the original portrait.
4. File as: `ModelSheet_Ally_Master.png`, `_Turnaround.png`, `_Heads.png`, `_Expressions.png`, `_Wardrobe.png`, `_Palette.png` in OneDrive `SAS/ModelSheets/Ally/`.

## The canon block (paste into every prompt)

```
CHARACTER CANON — Ally Quinn, age 26. True-crime podcaster and licensed private
investigator. Auburn/copper shoulder-length wavy hair with soft volume, swept
right-to-left across the brow. Large green-blue (teal) eyes, strong dark brows,
soft rounded face, light warm skin with subtle blush. Signature wardrobe, always
present unless stated: teal trench coat with wide lapels over a black
collared shirt; a small silver locket on a fine chain (holds her late father's
photo — hinge and seam visible); retro silver-and-black headphones worn around
the neck; brown leather satchel with podcast gear (worn cross-body).
Signature props: vintage chrome broadcast microphone; handheld cassette
recorder. Personality to convey in poses: sharp, observant, wry — a listener,
not a fighter.

STYLE: match the attached reference image exactly — stylized painterly
3D-illustration, oversized expressive eyes, clean silhouettes, rich saturated
color. IMPORTANT LIGHTING RULE: the blue/violet tones in the reference hair are
NOIR RIM-LIGHTING, not dyed streaks. In neutral light her hair is warm
auburn/copper throughout. Model sheet panels use FLAT NEUTRAL STUDIO LIGHTING
(soft even key, no colored rim) so true local colors are unambiguous — except
where a panel explicitly asks for the "noir lighting variant."
```

## Prompt 0 — Master model sheet (single page)

```
[CANON BLOCK]

Create a professional CHARACTER MODEL SHEET on a single flat light-grey
background (#DDDDDD) with clean layout and thin label lines, as used in
animation production. Landscape orientation. Contents, arranged clearly:

1. FULL-BODY TURNAROUND — the same standing pose repeated in four views:
   front, 3/4 left, left profile, back. Identical height and scale, feet on a
   shared ground line, arms relaxed. Neutral studio lighting.
2. HEAD DETAIL — one large 3/4 head study showing hair construction and eye
   detail.
3. EXPRESSION STRIP — three small heads: neutral, worried, surprised.
4. COLOR CALLOUTS — small labelled swatches sampled from the character: hair
   base, hair shadow, skin, blush, eyes, coat teal, shirt black, satchel brown,
   locket silver.
5. PROP INSET — the microphone and cassette recorder at relative scale to her
   hand.

Same character in every panel — identical face, proportions and wardrobe.
No background scenery, no text other than small neutral labels, no watermark.
```

## Prompt A — Full-body turnaround (dedicated, high resolution)

```
[CANON BLOCK]

Character turnaround sheet: ONE standing pose repeated in FIVE views — front,
3/4 left, left profile, 3/4 back, back — identical scale, evenly spaced on a
shared ground line, flat light-grey background. Neutral studio lighting, no
colored rim. Arms relaxed at sides, satchel worn cross-body (visible from
back), headphones around neck, trench coat closed. Show her full height;
proportions approximately 5.5 heads tall (stylized). Include a faint
horizontal guide line at head, shoulder, waist and knee height running behind
all five figures.
```

## Prompt B — Head turnaround

```
[CANON BLOCK]

Head-and-shoulders turnaround: the same head in five angles — front, 3/4 left,
left profile, 3/4 back, back — identical scale on one row, flat light-grey
background, neutral lighting. Hair construction must read clearly at every
angle: where the part sits, how the waves break over the right brow, the
length at the nape. Second row: the same front view twice — once with
headphones around the neck, once with headphones worn over the ears.
```

## Prompt C — Expression sheet (engine emotion set)

```
[CANON BLOCK]

Expression sheet: SEVEN identical head-and-shoulders busts in a grid, same 3/4
angle, same lighting, same crop — changing ONLY the facial expression. Label
each in small neutral text:
1 NEUTRAL — attentive, listening, faint knowing calm
2 HAPPY — warm genuine smile, eyes bright (rare for her; not giddy)
3 SAD — quiet grief, eyes down-tilted, mouth closed
4 ANGRY — cold precise fury, narrowed eyes, set jaw (never shouting)
5 SURPRISED — widened eyes, parted lips, brows high
6 WORRIED — knitted brows, tight mouth, eyes searching
7 CONFUSED — asymmetric brow, slight head tilt, pursed lips
These correspond to the game's portrait states and must be readable at
thumbnail size. Flat light-grey background.
```

## Prompt E — Palette & lighting conventions

```
[CANON BLOCK]

Two-part reference sheet on flat light-grey:
LEFT: the character bust in NEUTRAL studio light with a labelled swatch column
sampled from her: hair base (warm auburn/copper), hair shadow, skin, blush,
eye teal, coat teal, shirt black, satchel brown, locket silver.
RIGHT: the SAME bust in the game's signature NOIR LIGHTING — deep navy ambient
(#0A1220), warm amber key from the lower left, cool blue rim light on the hair
and shoulder — with a note: "Noir variant: blue tones are LIGHTING ONLY."
The two busts must be recognisably the identical character.

## Acceptance checklist (verify before a sheet becomes canon)

- [ ] Same face in every panel — if any view drifts (rounder cheek, different nose), regenerate that panel with the best panel as an added reference
- [ ] Hair reads warm auburn/copper in ALL neutral panels — no baked-in blue streaks
- [ ] Locket present and on a fine chain (not a pendant/necklace substitution)
- [ ] Headphones are retro over-ear, silver-and-black, resting around the neck
- [ ] Coat is teal with wide lapels — not green, not grey-blue
- [ ] Eyes green-blue/teal, not brown or pure blue
- [ ] Turnaround heights identical across views (overlay to check)
- [ ] Expression sheet emotions distinguishable at thumbnail size
- [ ] The noir-variant bust is identifiably the same person as the neutral bust
