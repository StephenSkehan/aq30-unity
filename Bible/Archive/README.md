# Bible Archive

Superseded bible versions are moved here — never deleted (preservation rule, 2026-07-11).

Convention: on each new export, move the outgoing file here with its era in the name,
e.g. `Ally Quinn Bible (v1 March 2026 scaffold).docx`.

**Pending (must be done on the OneDrive machine — the file was never committed):**
move `AllyQuinn Game/Bible/Ally Quinn Bible.docx` (the March 2026 scaffold) into this
Archive folder, renamed as above. Its content is preserved in the repo regardless, as
`Backups/bible_sources/march_scaffold.txt`.
